Integrasi Server

# Informasi Umum

Integrasikan aplikasi anda dengan sistem kami, untuk melakukan integrasi dengan sistem kami cukup mudah, ada beberapa hal yang harus anda lakukan, setelah melakukan pendaftaran, pastikan nomor rekening dan username serta password ibanking anda dan url penerima data sudah benar, pastikan juga API Key anda sudah dalam status aktif pada menu integrasi di website kami. Setiap ada transaksi baru yang masuk, data mutasi rekening akan kami kirimkan dalam bentuk JSON melalui url yang telah anda daftarkan dengan method POST.

Http Structure :

Field Output
Method POST
Format Data JSON

Data Structure :


{
" response_code " : " 0003 " ,
" response_details " : " Integrasi Data " ,
" bank " : " BCA " ,
" nomor_rekening " : " 012345678901 " ,
" data " : [
{
" tgl_transaksi " : 1,
" tipe " : " DEBIT " ,
" nominal " : " 50000 " ,
" deskripsi " : " TSRF E-Banking " ,
}
]
}

Processing Data : (Example)


public function take_data(Request $request) {
$data = new mutasi ;
$data->response_code = $request->response_code ;
$data->response_deskripsi = $request->response_details ;
$data->bank = $request->bank ;
$data->rekening = $request->nomor_rekening ;
$data->tanggal_transaksi = $request->data [" tgl_transaksi "] ;
$data->tipe = $request->data [" tipe "] ;
$data->jumlah = $request->data [" nominal "] ;
$data->deskripsi = $request->data [" deskripsi "] ;
$data->save() ;
}
