Riwayat Transaksi

# Informasi API

Informasi Request Http :

Http Method POST
URL http://localhost:8000/api
Path /mutasi
Format Data JSON

Structure Param Request :

Bidang Tipe Data Deskripsi Contoh
api_key String Terdapat pada menu integrasi di website abcdfghijkl
api_signature String Terdapat pada menu integrasi di website zyxwcklsds
rekening String Nomor rekening harus sudah didaftarkan pada website mutasi 012345678901
tgl_awal Date Gunakan format tanggal yyyy-mm-dd 2001-01-16
tgl_akhir Date Gunakan format tanggal yyyy-mm-dd 2001-01-30

# Request & Response

Request :

PHP - cURL
$curl = curl_init() ;
curl_setopt_array ( $curl, array (
CURLOPT_URL => ' http://localhost:8000/api/mutasi ',
CURLOPT_RETURNTRANSFER => true,
CURLOPT_ENCODING => '',
CURLOPT_MAXREDIRS => 10,
CURLOPT_TIMEOUT => 0,
CURLOPT_FOLLOWLOCATION => true,
CURLOPT_HTTP_VERSION => CURL_HTTP_VERSION_1_1,
CURLOPT_CUSTOMREQUEST => ' POST ',
CURLOPT_POSTFIELDS => array (
'api_key' => {key_anda},
'api_signature' => {signature_anda}
),
));

$response = curl_exec($curl);
curl_close($curl);
echo $response;
Jika anda ingin mencari riwayat transaksi pada nomor rekening tertentu , tambahkan field rekening pada bagian CURLOPT_POSTFIELDS => menjadi :
CURLOPT_POSTFIELDS => array (
' api_key ' => ' abcdfghijkl ',
' api_signature ' => ' zyxwcklsds ',
' rekening ' => ' 012345678901 '
),
Jika anda ingin mencari riwayat transaksi berdasarkan tanggal , tambahkan field tgl_awal dan tgl_akhir pada bagian CURLOPT_POSTFIELDS => menjadi :
CURLOPT_POSTFIELDS => array(
' api_key ' => ' abcdfghijkl ',
' api_signature ' => ' zyxwcklsds ',
' rekening ' => ' 012345678901 '
' tgl_awal ' => ' 2001-01-20'
' tgl_akhir ' => ' 2001-01-29'
),
